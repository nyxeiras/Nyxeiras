## What I’m Building

### SlimeChat

**SlimeChat** is the main project I’m currently pushing forward.

It’s being designed as a modern communication platform for gamers, inspired by the best parts of community chat, media sharing, clipping culture, and real-time interaction.
The goal is not just to make another chat app, but to build something that feels closer to a full gaming hub.

Planned direction includes:

- real-time messaging and community spaces
- a stronger desktop experience
- a true mobile application replacing the current PWA-based approach
- a browser version
- media-focused features inspired by clipping and sharing platforms
- future integration with a wider Slime ecosystem

At the moment, the clipping system and built-in editor are still in development and not yet implemented.

### SlimeTV

**SlimeTV** is an experimental system project built around reviving an old **Surface 2** and turning it into something closer to a lightweight media-focused platform.

The idea is to create a simple, living-room-friendly experience, similar in spirit to an **Android TV-style system**, with automatic updates and optional connection to SlimeChat.

The long-term direction includes:

- a custom interface adapted for media use
- automatic updates
- lightweight deployment for older hardware
- optional SlimeChat connectivity
- message notifications displayed in the corner of the screen
- the ability to disable those notifications when needed

This project is still exploratory, so its scope and timeline are not fully locked yet.
