# Main Projects

## SlimeChat

> A gaming-first communication platform.

**SlimeChat** is currently my main focus.

It’s being built as a modern alternative in the space of community chat, media sharing, and gamer-focused interaction.
The idea is to mix the best parts of real-time communication, clipped content, social presence, and platform connectivity into something bigger than a traditional chat app.

### Direction

- real-time messaging and community spaces
- desktop application
- browser version
- transition from PWA mobile setup to a true mobile app
- connected ecosystem features
- media and clip-oriented workflows
- long-term integration with other Slime products

## SlimeTV

> Reviving old hardware into a connected media platform.

**SlimeTV** is an experimental OS-style project made to give a second life to an old **Surface 2**.

The goal is to transform it into something close to an **Android TV-like experience**, but tailored to my own ecosystem, with automatic updates and optional integration with SlimeChat.

### Direction

- custom TV / media-oriented interface
- lightweight experience for older hardware
- automatic updates
- SlimeChat connectivity
- on-screen message notifications in the top-right corner
- ability to disable notifications when needed
- a practical “useful again” future for aging devices
